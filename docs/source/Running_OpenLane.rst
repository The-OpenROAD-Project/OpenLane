Running a OpenLane
==================

To start your docker run `make mount`

simply run the design:
.. code-block:: console
    make mount
    /flow.tcl -design <spm> -tag <run1>

Output of the design structure:

designs/<design_name>
├── config.tcl
├── runs
│   ├── <tag>
│   │   ├── config.tcl
│   │   ├── {logs, reports, tmp}
│   │   │   ├── cts
│   │   │   ├── signoff
│   │   │   ├── floorplan
│   │   │   ├── placement
│   │   │   ├── routing
│   │   │   └── synthesis
│   │   ├── results
